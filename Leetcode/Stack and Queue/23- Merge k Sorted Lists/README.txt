给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

 

示例 1：

输入：lists = [[1,4,5],[1,3,4],[2,6]]
输出：[1,1,2,3,4,4,5,6]
解释：链表数组如下：
[
  1->4->5,
  1->3->4,
  2->6
]
将它们合并到一个有序链表中得到。
1->1->2->3->4->4->5->6
示例 2：

输入：lists = []
输出：[]
示例 3：

输入：lists = [[]]
输出：[]
 

提示：

k == lists.length
0 <= k <= 10^4
0 <= lists[i].length <= 500
-10^4 <= lists[i][j] <= 10^4
lists[i] 按 升序 排列
lists[i].length 的总和不超过 10^4

思路：两种解法：第一种使用优先队列（最小堆），将所有链表的头先推入队列，然后每次推出最小的并连入新建链表（答案），加入最小的next节点，以此类推，最后将所有节点全部从小到大连在一起。
               第二种使用递归的思路，先将前两个链表merge，得到的结果再和第三个链表merge，直至和最后一个链表merge。
